# SENAVANCE - Sistema de Competitividad y Desarrollo Tecnologico

## SENA - Servicio Nacional de Aprendizaje (Colombia)

---

## 1. Que es SIGER / SENAVANCE

SENAVANCE es un sistema web interno del **SENA** (Servicio Nacional de Aprendizaje de Colombia) diseÃ±ado para la **gestion integral de usuarios, roles, planes y proyectos** de competitividad y desarrollo tecnologico a nivel nacional.

El sistema permite administrar la estructura organizacional del SENA (33 Regionales y 117+ Centros de Formacion), gestionar usuarios con roles jerarquicos, formular y dar seguimiento a proyectos, y generar reportes institucionales.

### Usuarios objetivo

- Administradores del sistema (nivel ROOT/NACIONAL)
- Coordinadores regionales
- Funcionarios de centros de formacion
- Contratistas y personal de planta del SENA
